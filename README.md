# Field Day Mobile Data Collector

A Progressive Web App for collecting field research data (snake, mammal, arthropod, amphibian entries) used by the ASU Field Day project. Built with Create React App, Firebase Authentication, and Firestore with offline persistence.

## Quick start

```bash
npm install
cp .env.example .env.local   # then fill in values from the Firebase Console
npm start
```

See [CONTRIBUTING.md](CONTRIBUTING.md) for full setup, deployment, and contribution details.
